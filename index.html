필요한 건 여섯 개의 숫자뿐.

복잡한 장식은 덜어내고, 추첨 순간과 결과가 또렷하게 보이도록 구성했습니다.

Secure random 지원되는 브라우저에서는 crypto 기반 난수를 사용합니다.
One set 1부터 45까지 중복 없이 여섯 번호를 정렬해 보여줍니다.
Supabase sync Vercel 환경변수로 연결된 Supabase에 추첨 결과를 저장합니다.
